# ExamSys
[English Version Click Here](README_en.md)

## 项目简介
本项目是一个基于Qt5.15.2的考试系统，用于管理考试信息。系统包括登录、考试信息管理、考试安排等功能。

## 项目结构
项目包含以下文件和目录：

```shell
D:.
│   account.txt
│   dialog.cpp
│   exam.txt
│   examdialog.cpp
│   examdialog.h
│   ExamSys.pro
│   ExamSys.pro.user
│   image.qrc
│   login.ico
│   login.png
```
